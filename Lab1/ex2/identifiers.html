Identifier

  1. Student ID (short for identification), or also called student URN (unique reference number) which is a number given to all university students that confirms their attendance at their given university.

    2. Eg: 20194571

  2. An account number is a set of digits used to identify your bank account. Each bank account has its own unique numerical code. You'll need your account number and sort code to send and receive money.

    3. Eg: 103870480387

  3. Alias is a personalized name for your account to be displayed instead of the account number. Use names like "Main account", "Partner's account EUR"t or name of a person/company to more easily recognize them in the Internet Bank

    4. Eg: HOANG0202

  4. A Global Trade Item Number (GTIN) is a unique and internationally recognized identifier for a product. When a GTIN is available, it will appear next to the barcode on your product's packaging or book cover. Submit GTINs in your product data using the GTIN [gtin] attribute to help us classify and display your products.

  5. The Citizen identity card number is embedded with the main personal identification number, or can be understood as a separate code associated with the citizen in the national database system. The Ministry of Public Security will assign each citizen a unique personal identification number.
